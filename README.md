# 🚀 Shortest Path Visualizer (NetworkX)

A Python-based project to visualize the shortest path between nodes in a graph using NetworkX and Matplotlib.

---

## 📌 Features

* 📊 Graph creation using user input
* 🔍 Shortest path calculation using Dijkstra’s Algorithm (via NetworkX)
* 🎯 Path length display
* 🎨 Graph visualization with highlighted shortest path

---

## 🛠️ Tech Stack

* Python
* NetworkX
* Matplotlib

---

## ⚙️ How It Works

1. User inputs graph edges (with or without weights)
2. Graph is created using NetworkX
3. Shortest path is computed using Dijkstra’s Algorithm
4. The graph is visualized:

   * Nodes → Light blue
   * Edges → Gray
   * Shortest path → Red

---

## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/yaswanth-AIML/shortest-path-visualizer-networkx.git
   ```

2. Navigate to the project folder:

   ```bash
   cd shortest-path-visualizer-networkx
   ```

3. Install dependencies:

   ```bash
   pip install networkx matplotlib
   ```

4. Run the project:

   ```bash
   python main.py
   ```
